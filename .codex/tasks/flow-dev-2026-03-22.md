# Task Brief: Flow Dev 2026-03-22

## 작업명

- 오버레이 액션 정규화 + 스테이지 선택 보강 + 튜토리얼 1회성 저장 + 결과 보상 강화

## 목표

- `flow-development-2026-03-21.md`에 정의된 P1·P2 항목을 구현한다.
- 플레이어가 실패 후 홈으로 돌아갈 수 있게 한다.
- 스테이지 선택 전 목표·튜토리얼 여부·테마·추천 흐름을 더 명확히 제공한다.
- 같은 스테이지를 반복 플레이할 때 이미 본 튜토리얼을 다시 보여 주지 않는다.
- 클리어 오버레이에서 별 획득과 다음 해금 정보를 더 명확히 보여 준다.

## 구현 범위

### P1-A: 오버레이 액션 규칙 정규화 (gameplay.gd)

- 실패 오버레이에 `홈으로` 2차 버튼을 추가한다.
- `_on_overlay_secondary_button_pressed()`에서 `restart_stage` 액션 처리를 추가한다.
- 사용하지 않는 `next_stage` 2차 버튼 핸들러를 제거한다.

### P1-B: 스테이지 선택 상세 정보 보강 (stage_select.gd)

- 선택 스테이지 타이틀에 `[튜토리얼]` 배지를 추가한다.
- 선택 스테이지 본문에 최고 별 기록을 노출한다.
- 선택 스테이지 본문에 테마 키 정보를 노출한다.
- 선택 스테이지 본문에 현재 추천 스테이지 여부를 표시한다.

### P2-A: 튜토리얼 1회성 저장 (game_session.gd + gameplay.gd)

- `game_session.gd`에 `seen_tutorial_stage_ids` 배열을 추가한다.
- `mark_tutorial_seen(stage_id)`, `is_tutorial_seen(stage_id)` 헬퍼를 추가한다.
- `gameplay.gd`의 `_setup_tutorial()`에서 이미 본 튜토리얼은 건너뛴다.

### P2-B: 결과 오버레이 보상 정보 강화 (gameplay.gd)

- 클리어 오버레이 본문에 다음 스테이지 해금 정보를 추가한다.
- 별 획득 텍스트를 더 눈에 띄는 위치로 이동한다.

## 완료 조건

- 실패 오버레이에서 `홈으로` 버튼을 눌러 홈 화면으로 돌아갈 수 있다.
- 스테이지 선택 화면에서 선택한 스테이지의 튜토리얼 여부, 최고 기록, 테마를 확인할 수 있다.
- Stage 1을 두 번 진입했을 때 두 번째 진입에서는 튜토리얼 오버레이가 나타나지 않는다.
- 클리어 오버레이에서 별 등급과 다음 스테이지 해금 여부를 확인할 수 있다.
