# PM Team 템플릿

## 팀 목표

- 전체 목표, 범위, 일정, 우선순위, 승인 기준을 관리한다.

## 기본 구성

- `PM Lead`
- `Priority Manager`
- `Risk Manager`
- `Delivery Coordinator`

## 팀장 책임

1. 작업 브리프를 만든다.
2. 어떤 팀이 무엇을 먼저 해야 하는지 정한다.
3. 승인 기준과 제외 범위를 고정한다.
4. 팀 간 충돌을 정리한다.

## 서브 에이전트 책임

- `Priority Manager`
  - 우선순위와 마일스톤 정리
- `Risk Manager`
  - 일정/범위/품질 리스크 정리
- `Delivery Coordinator`
  - 핸드오프 상태와 통합 진행 상황 정리

## 추천 모델

- PM Lead: `gpt-5.2` / `high`
- PM Members: `gpt-5.4-mini` / `medium`
