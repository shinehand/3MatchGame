# Implementation Review 2026-03-18

## 의도 대비 확인

목표 의도:

- `캔디크러시사가 계열`의 모바일 3매치 퍼즐
- `귀여운 동물 블록` 테마
- `게임다운 UX`
- `난이도별 스테이지`를 만들 수 있는 준비

## 현재 맞게 구현된 부분

- 동물 블록 5종이 실제 보드에 연결됨
- 유효 스왑, 매치, 제거, 낙하, 리필, 연쇄가 동작함
- 스와이프 입력이 가능함
- Stage 시작/클리어/실패 오버레이가 있음
- 목표, 이동 수, 점수, 난이도 표시가 HUD에 연결됨
- Stage 데이터가 `StageCatalog`로 분리되어 난이도별 확장이 가능함
- 4매치 줄 제거, 5매치 폭발 특수 블록이 들어갈 구조가 구현됨
- 배경과 매치 버스트 이펙트에 귀여운 컨셉 이미지를 연결할 수 있는 구조가 생김
- 덤불 장애물과 덤불 목표가 스테이지 데이터 및 HUD 목표 칩에 연결됨
- 클리어 등급과 실패 힌트가 오버레이에 반영됨

## 아직 부족한 부분

- 장애물이 없어 레벨 개성이 부족함
- Stage 선택 화면과 메타 진행이 없음
- 손맛을 강화할 사운드/진동이 없음
- 튜토리얼이 최소 수준이라 첫 플레이 가이드가 약함

## 추가 기획 판단

현재 의도에 맞추려면 아래 기획을 더 확정해야 한다.

1. `장애물 1종`의 제거 규칙
2. `스테이지 클리어 보상` 구조
3. `레벨 선택 화면`과 난이도 곡선
4. `튜토리얼 안내`를 실제 플레이 흐름으로 강화
5. `사운드/햅틱` 기준

## 개발 우선순위 제안

1. 튜토리얼 안내 강화
2. 승리/실패 연출 강화
3. 스테이지 선택 화면
4. 저장과 진행도
5. 사운드/햅틱
