# 100 Stage Progression Plan

## 원칙

- 현재 메커닉만 쓴다: `8x8 마스크 보드`, `수집`, `점수`, `덤불`, `4매치 줄 제거`, `5매치 3x3 폭발`
- 새 메커닉은 100스테이지 안에 넣지 않는다.
- 난이도는 `목표 조합`, `보드 모양`, `덤불 밀도`, `이동 수`, `목표 수치`로만 올린다.

## 밴드 요약

- `1-10`: 정글 입구 구조, 단일 수집, 넓은 보드, 덤불 0-2, `moves 12-13`
- `11-20`: 덫 흔적 발견, 복수 수집 + 첫 점수 목표, 얕은 구멍, 덤불 1-4, `moves 13-14`
- `21-30`: 야영지 외곽 진입, 덤불 관리 시작, 비대칭 첫 진입, 덤불 4-6, `moves 14-15`
- `31-40`: 복수 동물 구조, 수집 2종 + 점수 + 덤불, 복합 루트, 덤불 6-7, `moves 15-16`
- `41-50`: 강가 구역, 점수 중심 압박, 좁은 보드, 덤불 8-9, `moves 16-17`
- `51-60`: 야영지 내부, 긴 경로 + 고밀도 장애물, 덤불 9-10, `moves 16-17`
- `61-70`: 깊은 밀림, 전문가 압박, 협소한 길목, 덤불 10-11, `moves 17-18`
- `71-80`: 탈출 준비, 마스터리 테스트, 읽기 어려운 마스크, 덤불 11-12, `moves 18`
- `81-90`: 코끼리 수송 경로 추적, 피날레 진입, 최고 복합 목표, 덤불 12-13, `moves 18`
- `91-100`: 코끼리 최종 구출, 가장 좁은 루트, 덤불 13-14, `moves 18-19`

## 고정 학습 해금

- `Stage 3`: 4매치 학습
- `Stage 7`: 5매치 학습
- `Stage 11`: 점수 목표 정식 도입
- `Stage 15`: 덤불이 본격 난이도 요소가 됨
- `Stage 25`: 비대칭 마스크가 흔해짐
- `Stage 45`: 좁은 게이트형 마스크가 표준이 됨
- `Stage 65`: 전문가 루트 판독 구간
- `Stage 85`: 최종권 마스크 압박
- `Stage 95`: 피날레형 최종 압박

## 스토리 마일스톤

- `Stage 10`: 정글 입구 첫 구조 완료
- `Stage 20`: 밀렵꾼 덫 구역 해제
- `Stage 30`: 야영지 외곽 우리 정리
- `Stage 50`: 코끼리 흔적 발견
- `Stage 70`: 대형 우리 외곽 구역 도달
- `Stage 90`: 코끼리 우리 앞 봉쇄선 정리
- `Stage 100`: 거대한 코끼리 구출

## 운영 규칙

- 각 밴드의 마지막 1-2개 스테이지는 다음 밴드의 규칙을 미리 보여준다.
- 초반은 실패보다 성공률을 우선한다.
- 후반은 새로운 규칙 추가보다 기존 규칙의 조합 난이도를 올린다.
- 100스테이지 이후가 아니라면 메커닉 추가는 보류한다.
