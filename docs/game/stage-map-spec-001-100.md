# Stage Map Spec 001-100

## Legend

- `col(r10)` = 토끼 10개 수집
- `score(2500)` = 점수 2500 달성
- `bush6` = 덤불 6개 제거 목표
- `moves 12-13` = 이동 수 밴드
- shape family = 보드 마스크 계열
- 마지막 메모는 `story beat`를 함께 포함할 수 있다.

## 1-10

### 챕터 목표

- 정글 입구에서 작은 동물들을 처음 구조한다.
- 플레이어가 `동물을 구출한다`는 캠페인 의미를 이해하는 구간이다.
