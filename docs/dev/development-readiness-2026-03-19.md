## 개발 입력 대기 항목

- 튜토리얼 문구와 트리거 명세
- 설정 화면 상태 정의
- 자산 패키지별 파일명과 해상도
- 밴드별 배경 키

## 내부 리뷰 포인트

- portrait/landscape 분기에서 버튼 의미가 일관적인지
- 자산 교체가 씬 구조 개편 없이 가능한지
- 게임플레이 중 오버레이와 상태 카드 충돌이 없는지
- 대표 스테이지에서 난이도와 시각 정보가 동시에 깨지지 않는지
