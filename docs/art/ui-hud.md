# UI HUD Spec

## 목적

landscape 기준 플레이 화면의 HUD와 기본 UI 시각 방향을 정의한다.

## 화면 구조

- 중앙 좌측: 퍼즐 보드
- 우측 세로 패널: 목표, 남은 이동 수, 점수
- 상단 바: 스테이지 번호, 일시정지 버튼

## 우선순위

1. 남은 이동 수
2. 목표 진행도
3. 보드 가독성
4. 점수

## HUD 스타일

- 패널은 둥근 카드형
- 배경색은 하늘색/민트 계열
- 숫자는 굵고 크며, 텍스트보다 먼저 읽혀야 함
- 목표 아이콘은 동물 얼굴 썸네일과 숫자 조합으로 표시

## 버튼 스타일

- 기본 버튼: 노랑 또는 핑크 강조색
- 보조 버튼: 하늘색 또는 흰색 카드형
- 눌림 상태: 4~8px 아래로 눌린 듯한 음영 변화

## 플레이 화면 필수 요소

- 스테이지 번호
- 남은 이동 수
- 현재 목표 1~2개
- 일시정지 버튼
- 점수

## 레이아웃 가이드

- 보드가 화면에서 가장 먼저 보여야 함
- 우측 HUD 폭은 전체 너비의 약 `22%~28%`
- 상단 바 높이는 화면 높이의 약 `10%` 내외
- 버튼 최소 터치 영역은 충분히 크게 유지

## 이펙트 방향

- 매치 성공 텍스트: `POP`, `COMBO`, `GREAT` 같은 짧은 단어
- 점수 상승 숫자는 위로 떠오르며 빠르게 사라짐
- 목표 달성 직전에는 목표 카드가 짧게 강조됨
