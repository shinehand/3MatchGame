# Current Art Audit 2026-03-18

## 총평

- 현재 버전은 `블록 중심 퍼즐 프로토타입`으로는 맞다.
- 하지만 아트 문서가 의도한 `스티커형 동물 세계관 + 명확한 캐주얼 UX`에는 아직 못 미친다.

## 핵심 문제

### 1. UI가 기본 위젯 상태다

- 버튼, 패널, 스테이지 카드에 세계관이 없다.
- 눌림 상태, 볼륨감, 강조색 체계가 없다.

### 2. 정보 위계가 약하다

- 보드보다 텍스트 카드와 개발용 버튼이 먼저 보인다.
- 제목, 부제, 상태 텍스트, 팁 카드가 동시에 경쟁한다.

### 3. 타일 상태 표현이 읽기 어렵다

- 선택 glow가 얼굴을 씻어버린다.
- 특수 블록이 유니코드 문자라 세계관과 분리된다.

### 4. 홈과 스테이지 선택이 약하다

- 홈은 큰 블록 이미지를 얹은 개발용 레이아웃에 가깝다.
- 스테이지 선택은 텍스트 버튼만 있어 보상감이 없다.

### 5. 색감과 리소스 층이 부족하다

- 동물 블록은 저대비, 배경은 흐림, 장애물도 뿌옇다.
- UI 프레임, 특수 배지, 클리어/실패 얼굴, 스테이지 카드 비주얼이 없다.

## 바로 필요한 아트 작업

1. `UI 테마 팩`
2. `특수 블록/선택/장애물` 배지 세트
3. `홈 화면 마스코트`와 `스테이지 카드`
4. `클리어/실패 오버레이` 감정형 리소스
5. 배경 대비와 전체 색 체계 재정리

## 실행 기준

- 버튼은 눌렀을 때 버튼처럼 보여야 한다.
- 목표, 이동 수, 다음 행동이 첫 시선에서 읽혀야 한다.
- 블록 얼굴은 어떤 오버레이가 와도 알아볼 수 있어야 한다.
- 배경은 분위기만 주고 보드 가독성을 침범하면 안 된다.
