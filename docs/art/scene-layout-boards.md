# Scene Layout Boards

## 목적

- 생성형 풀 일러스트 대신 실제 구현에 바로 연결되는 `레이아웃 보드`를 제공한다.
- 모든 보드는 `동물 블록 에셋`을 중심으로 씬 구조를 잡는다.

## 1. 홈 화면 레이아웃

- 보드 파일: [home_layout_board.svg](/Users/shinehandmac/Documents/puzzle/assets/concepts/home_layout_board.svg)

핵심:

- 상단 로고
- 중앙 `PLAY`
- 좌우 실제 블록 얼굴 기반 마스코트
- 배경은 조용한 하늘색

## 2. 플레이 화면 레이아웃

- 보드 파일: [gameplay_layout_board.svg](/Users/shinehandmac/Documents/puzzle/assets/concepts/gameplay_layout_board.svg)

핵심:

- 왼쪽 대형 보드
- 오른쪽 HUD 카드 열
- 목표 칩은 실제 동물 얼굴 축소판 사용
- 배경은 가독성 우선

## 3. 오버레이 UI 보드

- 보드 파일: [overlay_ui_board.svg](/Users/shinehandmac/Documents/puzzle/assets/concepts/overlay_ui_board.svg)

핵심:

- 시작, 클리어, 실패 팝업의 구조 분리
- 클리어와 실패 모두 같은 세계관의 얼굴 표정 사용
- 카드형 UI와 큰 CTA 버튼 유지

## 4. 홈용 마스코트 얼굴 기준

- 보드 파일: [mascot_face_board.svg](/Users/shinehandmac/Documents/puzzle/assets/concepts/mascot_face_board.svg)

핵심:

- 홈 마스코트는 실제 블록 얼굴을 확대/확장한 버전이어야 한다.
- 전신 포스터형 캐릭터는 사용하지 않는다.
- Stage 1과 홈 화면의 대표 마스코트는 토끼를 우선한다.
