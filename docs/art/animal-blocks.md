# Animal Blocks Spec

## 목적

매치3 보드에서 사용하는 기본 동물 블록 5종의 시각 방향을 정의한다.

## 공통 규칙

- 각 블록은 `정면 얼굴` 기준
- 작은 화면에서도 0.5초 안에 구분 가능해야 함
- 외곽선은 너무 얇지 않게 유지
- 눈은 크게, 입은 단순하게
- 하이라이트는 상단, 그림자는 하단 위주로 넣어 말랑한 볼륨감 표현

## 블록 공통 구조

- 캔버스: `256x256`
- 안전 영역: 가장자리에서 최소 `20px`
- 기본 형태: 둥근 사각형 타일 안에 얼굴 삽입
- 상태:
  - 기본
  - 선택됨
  - 매치 직전 강조
  - 매치 성공 표정

## 1. 토끼

- 메인 색: 밝은 분홍 + 크림톤
- 형태 포인트: 긴 귀, 작은 코, 웃는 입
- 인상: 가장 친근하고 대표적인 캐릭터
- 용도: 튜토리얼과 메인 홍보용 키 비주얼 우선 후보

## 2. 곰

- 메인 색: 카라멜 브라운
- 형태 포인트: 둥근 얼굴, 작은 귀, 넓은 볼
- 인상: 안정감 있고 묵직한 느낌

## 3. 고양이

- 메인 색: 오렌지 또는 크림 오렌지
- 형태 포인트: 삼각 귀, 반달 눈, 작은 수염 점
- 인상: 장난스럽고 반응이 잘 보이는 캐릭터

## 4. 병아리

- 메인 색: 선명한 노랑
- 형태 포인트: 작은 부리, 동그란 눈, 가장 단순한 실루엣
- 인상: 작고 경쾌한 포인트 캐릭터

## 5. 개구리

- 메인 색: 민트 그린
- 형태 포인트: 튀어나온 눈, 넓은 입, 볼이 넓은 얼굴형
- 인상: 실루엣 구분이 가장 쉬운 변형형 캐릭터

## 매치 피드백 방향

- 선택 시: 살짝 확대 + 밝은 테두리
- 스왑 시: 짧은 탄성 이동
- 매치 성공 시: 눈 감기, 웃는 입, 1프레임 찌그러짐 후 팝
- 연쇄 시: 캐릭터 주변에 별/하트/반짝이 중 하나 선택 적용
