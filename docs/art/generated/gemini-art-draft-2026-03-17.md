## 요약

- 실행 도구: `gemini-art.sh`
- 목적: Animal Pop Match용 동물 블록과 HUD 컨셉 초안 생성
- 결론: `2D 스티커 스타일` 안이 현재 프로젝트 방향과 더 잘 맞음

## 후보 A

### 방향

- Soft Claymorphism
- 말랑한 질감의 3D처럼 보이는 2D 스타일

### 장점

- 터치하고 싶은 질감이 강함
- 개별 블록 존재감이 큼

### 단점

- 현재 프로젝트의 `순수 2D 스프라이트` 방향과 어긋남
- 제작과 일관성 유지 비용이 더 커질 수 있음

## 후보 B

### 방향

- 굵은 외곽선의 2D 벡터 + 스티커형 흰 테두리
- 순수 2D 스프라이트 스타일

### Gemini 핵심 출력

- 비주얼 방향: 굵고 깨끗한 외곽선의 2D 벡터 일러스트, 종이에서 떼어낸 듯한 스티커 느낌
- 색상 키워드: Vibrant Pastels, High Contrast, Candy Colors
- UI 무드: 둥근 모서리, 직관적인 아이콘, 가독성 높은 굵은 폰트
- 배경 아이디어: 연한 체크무늬나 도트 패턴이 들어간 파스텔 톤 숲/정원

### 동물 블록 포인트

- 토끼: 연분홍, 긴 귀, 큰 눈
- 곰: 따뜻한 갈색, 둥근 귀, 코 주변 밝은색
- 고양이: 주황색, 뾰족한 귀, 수염
- 병아리: 밝은 노랑, 작은 부리, 단순한 형태
- 개구리: 라임 그린, 큰 눈, 넓은 입

### HUD 포인트

- 상단: 레벨, 점수, 목표 게이지, 일시정지
- 사이드: 아이템 슬롯
- 중앙: 8x8 또는 9x9 보드
